\abstract{\addtocontents{toc}{\vspace{1em}} % Add a gap in the Contents, for aesthetics

Hippocampus is a structure which is crucial for spatial processing and episodic memory. The mechanisms of episodic memory are unclear. It has been proposed that spatial processing provides the basis for encoding episodic memories. In this study, the spatial processing in rodents was the main focus. Exploratory data analysis was conducted to characterize place cell activity in CA3 and CA1 at single cell level and ensemble dynamics in the whole population, with the effort to evaluate the interaction between internal dynamics and the external stimuli, and its influence on the mechanisms of spatial processing. 
}
